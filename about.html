About This Project

This project provides a HMRC-style receipt extractor dashboard designed for self-employed individuals to maintain professional, compliant expense records.

Features include:

  • OCR extraction of receipts using Tesseract.js.
  • Manual entry option for unscannable receipts.
  • Duplicate detection to prevent double entries.
  • HMRC checks with green ticks/red crosses for compliance.
  • Running totals for Net, VAT, and Gross.
  • CSV export and PDF reports with charts.
  • Dashboard charts (category breakdown + monthly totals).

Disclaimer: This tool is provided for assistance but does not replace formal accounting advice. Always confirm claims with a qualified accountant for HMRC compliance.
