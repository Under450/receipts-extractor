Setup Instructions

  1. Download the ZIP and unzip it on your iPhone using the Files app.
  2. In your GitHub repository, tap Code → Upload files.
  3. Select all of the following files and folders:
    • index.html
    • rules.html
    • instructions.html
    • about.html
    • css/style.css
    • js/app.js
  4. Commit the changes.
  5. Go to Settings → Pages and set the source to: Main branch, root folder.
  6. Save. Your live site will be available at: https://yourusername.github.io/your-repo-name/
