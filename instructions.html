GitHub/iPhone Setup Instructions

Follow these steps to upload and view your HMRC Receipt Extractor site:

  1. Download the ZIP and unzip on your iPhone (Files app).
  2. Go to your GitHub repo → Code → Upload files.
  3. Select index.html, rules.html, instructions.html, about.html.
  4. Commit changes.
  5. Go to Settings → Pages → Source: main branch, Folder: /(root).
  6. Wait 1–2 minutes for GitHub Pages to publish.
  7. Refresh the Pages screen → copy or tap your live site link.
  8. Add to iPhone Home Screen for quick access.
